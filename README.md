# Clinical-Analytics-Learning
.🏥 Healthcare Data Analytics | Transforming Clinical Data into Actionable Insights

A Python-based tool that generates structured clinical study codes using hospital name and medical specialty.

---

## 🚀 Problem Statement
In clinical research and healthcare systems, studies and data need standardized naming for identification and tracking.

This project simulates how real-world clinical codes are generated.

---

## 💡 How It Works
- Takes hospital/institute name as input  
- Takes research focus/specialty  
- Converts them into short codes  
- Generates a structured identifier with year and unique number  

---

## 🧪 Example

**Input:**
RML, Cardiology  

**Output:**
RML-CAR-2026-583  

---

## ⚙️ Tech Stack
- Python  
- String Manipulation  
- Random Module
